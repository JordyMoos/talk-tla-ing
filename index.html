What is TLA+

TLA+ is a language for high level modeling of digital systems. (Algorithms and programs)

TLA+ is used to model critical parts of those systems.

It is used to help to find and correct design errors.

With TLA+ you create a blueprint of your digital system which you are able to test for correctness.

If we would be describing the hours of a clock then we could say that valid valid are all integers from 1 till 12

And that the next valid value of a hour is the previous hour + 1. Unless current hour is 12 then it should be one

While this can be described in pseudo code + english. The down side of doing so is that you algorithm can not be tested for correctness

It gets even more complicated if you want to specify how the spaceships thrusters work when initiating a moon lander.
